## 🚀 Key Features

### 🎲 Number Guessing Game
- Generated a random number
- Allowed multiple guessing attempts
- Used `while` loop for iteration
- Implemented `break`, `continue`, and `else`
- Displayed user-friendly messages based on guesses

### 🔢 Multiplication Table Generator
- Accepted user input
- Generated multiplication tables from 1 to 10
- Implemented using `for` loop and `range()`

### ⚖️ BMI Calculator
- Created a reusable function
- Accepted weight and height as input
- Calculated Body Mass Index (BMI)
- Displayed formatted BMI result
